# RAP CDS Views

This folder contains CDS Views used in the RAP layer.

## Implemented Views

- **Z_I_WAREHOUSE_STOCK** – Basic interface view for stock data
- **Z_I_WAREHOUSE_KPI** – Analytical view with KPIs (turnover, days of supply, ABC classification, slow/dead stock)

## Purpose
- Provide clean, reusable data models
- Enable real-time analytics
- Serve as foundation for Fiori Elements applications
