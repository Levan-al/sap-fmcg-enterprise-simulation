# RAP Services

This folder contains Service Definitions and Service Bindings for OData exposure.

## Components

- **Service Definition** (`Z_UI_WAREHOUSE_STOCK`)
  - Exposes CDS Views as OData entities

- **Service Binding** (`Z_UI_WAREHOUSE_STOCK_V4`)
  - OData V4 binding for Fiori Elements
  - Supports List Report + Object Page

## Purpose
- Enable modern UI development with Fiori Elements
- Allow consumption from Fiori Launchpad or external applications
- Support analytical queries and actions
