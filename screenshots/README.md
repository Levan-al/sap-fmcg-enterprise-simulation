## How to add screenshots
1. Take clear screenshots from SAP GUI / Fiori
2. Name them meaningfully (e.g. `z_stock_visibility_alv.png`)
3. Add short description in this README
4. Commit with message "Add screenshot: ..."

**Tip for senior review:** Good screenshots with proper data make the project much more convincing.
