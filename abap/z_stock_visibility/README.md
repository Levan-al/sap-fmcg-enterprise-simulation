# z_stock_visibility

Unified Stock Visibility Report for Warehouse Operations.

Features:
- Shows stock from MARD (plant, storage location, batch)
- Authorization check
- Field-Symbols and performance optimized logic
- ALV output with totals

- z_stock_visibility_top.abap
- z_stock_visibility_sel.abap
