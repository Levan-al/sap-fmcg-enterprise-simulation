# ABAP Custom Development

This folder contains all custom Z-reports and modular ABAP code for the FMCG enterprise simulation.

Current Z-reports:
- z_stock_visibility (Stock Visibility Report)
- z_goods_movement (Goods Movement Control)
- z_process_monitor
- z_delivery_control
- z_billing_engine
- z_audit_trail

Each Z-report is built with modular structure (TOP + SEL + MAIN) following clean ABAP best practices.
