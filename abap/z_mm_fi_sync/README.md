# z_mm_fi_sync

MM-FI Synchronization and Reconciliation Report.

Features:
- Compares Material Documents (MSEG/MKPF) with FI documents (BSEG/BKPF)
- Detects mismatches in amount, quantity, or account assignment
- Negative stock vs FI posting check
- Detailed error logging and reconciliation list
- Summary dashboard of integration issues

Purpose: Ensure consistency between MM and FI modules and prevent financial discrepancies.
